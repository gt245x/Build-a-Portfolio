Python

Programming Foundations with Python is the first computer programming course that will be studied in the full Stack Web Developer Nanodegree. Learning python is like climbing an uphill stairs. The course will discuss the use of functions and classes in learning object oriented programming. Some of the projects in this course include:

  • Project take a break
  • Project profanity editor
  • Project movie database

Relational database

Relational databases are powerful tools used throughout the industry and one of the backend developers most powerful tool. The course introduces the basics of SQL and how to connect a python code to a relational database. By the end of the course, the student should be able to organize and store application data in a relational database. The final project is building a database backed software to run a particular kind of game or tournament.

Linux Server

Linux is a Unix-like operating system assembled under the model of free and open-source software development and distribution. The course involves using a virtual machine to host apps and install updates via a baseline installation of a Linux distribution. The course also introduces working within a shell and running a vagrant virtual machine. The final project tests the students on preparing linux virtual machine to install updates, securing it from a number of attack vectors and configuring web and database servers.
